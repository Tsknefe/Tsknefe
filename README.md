## Featured Projects

### Crypto Trading Engine

FastAPI based algorithmic trading system designed around risk-adjusted decision making.

Features include technical indicator pipelines, confidence weighted signals, volatility aware position sizing, automated trade execution and portfolio state logging.

The system prioritizes capital preservation and risk adjusted performance rather than simple prediction accuracy.

---

### BankApiABP

Banking backend infrastructure built using ABP Framework and Domain Driven Design principles.

Includes modular domain services, transaction management, permission systems, concurrency handling and structured logging.

Designed as a production-style backend architecture.

---

### String Art Generator

Computer vision based generator that transforms images into string-art instructions using image processing techniques.
